# 🌹 Srishti Kumari - Portfolio Website

A modern, elegant, and visually stunning personal portfolio website built with React, Vite, TailwindCSS, and Framer Motion. Features a premium rose gold and black color theme inspired by "Apple meets Data Science" design philosophy.

## ✨ Features

- **Modern Tech Stack**: React 18, Vite, TailwindCSS, Framer Motion
- **Premium Design**: Rose gold (#C9747A) & black theme with elegant animations
- **Fully Responsive**: Optimized for all devices (mobile, tablet, desktop)
- **Smooth Animations**: Framer Motion scroll reveals, hover effects, and transitions
- **Typing Effect**: Dynamic hero section with react-typed
- **Interactive Components**: Animated navigation, project cards, skill bars
- **Contact Form**: Functional contact form with validation
- **SEO Optimized**: Semantic HTML and meta tags

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/srishti717/portfolio.git
cd react-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## 📦 Build for Production

```bash
npm run build
```

The optimized production build will be created in the `dist` folder.
